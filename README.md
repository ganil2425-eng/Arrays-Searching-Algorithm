# 2D Array Search in C

A simple C program to create a 2D array, take user input, and search for a specific element. It demonstrates nested loops, conditional checks, and flag logic, making it beginner‑friendly for learning arrays.

## Features
- Dynamic array size defined by user input  
- Input values stored in a 2D matrix  
- Search functionality to check if a given element exists  
- Clear output messages for found/not found results  

## Usage
1. Save the code in `array_search.c`  
2. Compile using:
gcc array_search.c -o array_search
3. Run:
./array_search
4. Follow the prompts to input array size, elements, and search value
